# 🚀 Mini CRM - Client Lead Management System

A complete full-stack CRM application with sales pipeline tracking, revenue analytics, and lead management.

## ✨ Features

- 🔐 **Secure Authentication** - JWT-based admin login
- 📊 **7-Stage Sales Pipeline** - New → Contacted → Qualified → Proposal → Negotiation → Won/Lost
- 🎯 **Drag & Drop Kanban** - Move leads through pipeline stages
- 💰 **Deal Value Tracking** - Track revenue for each lead
- 📈 **Analytics Dashboard** - Visual charts and insights
- 🎨 **Conversion Funnel** - Track lead progression with drop-off rates
- ✏️ **Full CRUD Operations** - Create, Read, Update, Delete leads

## 🛠️ Tech Stack

### Frontend
- React 18, Tailwind CSS, Chart.js

### Backend  
- Node.js, Express.js, MongoDB Atlas, JWT Authentication
